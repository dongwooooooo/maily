# AI 메일 비서 — 브랜드 컬러 확정

색상·상태 시스템의 확정 사항만 정리한다. 목업 실물은 `design/boards/v1/current/`(HTML), 요약은 루트 `DESIGN.md` 참조.

## 1. 베이스 팔레트

| 토큰 | 값 | 비고 |
|---|---|---|
| 배경 | `#FFFFFF` | 순수 화이트 |
| 보조면 | `#F6F4EF` | 클린 아이보리. 배지·칩·소형 장식 전용 — 사이드바·상세 패널 같은 대면적 배경엔 쓰지 않는다 |
| 선택 상태 면색 | `--teal-soft`(틸 4% 알파) | 카드 선택·hover 공용. 좌측 3px 틸 룰과 함께 선택을 표시 |
| 잉크(헤딩·강조) | `#26282B` | 중성. 순흑 금지 |
| 잉크(본문) | `#43464A` | 중성 소프트. 웜 잉크(갈색기) 금지 |
| CTA / Primary | `#14807A` | 스튜어드 틸. 버튼·링크·행동 라벨 단일톤 |
| Success | `#218358` | 상태 순간에만 |
| Warning | `#A16207` | 상태 순간에만 |
| Danger | `#CE2C31` | 상태 순간에만 |

유채색은 시맨틱 3색 + CTA 틸 외 추가 금지.
계정 배지·계정 구분은 아바타(프로필 사진 또는 계정별 뉴트럴 아이콘)로만 한다 — 색상 코딩 금지.

## 2. 투명도·상태 시스템

- 잉크 알파 토큰: `--hover-ink` 4% · `--label` 78%(구조 라벨: 계정명·발신자·카운트) · `--muted` 64%(순수 부가정보) · `--line` 12% · `--line-strong` 22% · `--control` 48%
- 상태 레이어: hover 4%, pressed 10~12%, disabled 38%(비활성 컨트롤 전용)
- 서페이스는 solid, 상태·보더·배지만 alpha. 알파 서페이스 중첩 금지
- 비활성 처리에 요소 `opacity` 금지 — 알파 색 토큰으로 지정
- 38%는 비활성 컨트롤 전용. 읽어야 할 콘텐츠는 64% 이상
- 시맨틱 틴트(10%) 위에 시맨틱 색 텍스트 금지 — 틴트 위 텍스트는 잉크, 색은 도트/아이콘으로. 틸 정보 배너만 예외적으로 틸 텍스트 허용(기존 요약 라벨 선례)
- 아이보리 면은 22% 알파 보더 필수

## 3. 접근성 체제

- WCAG AA를 목표로 채택 — 위반은 차단이 아니라 측정·보고
- 본문 4.5:1 / 대형·볼드 텍스트 및 UI 경계 3:1 / 비활성·부수 텍스트 예외
- 모든 대비는 알파 합성 후 색 기준으로 계산

## 4. 브랜드 톤

- 포지션: Gmail 위에 얹히는 조용한 브리핑 레이어
- 형용사: 차분한·정확한·투명한·정갈한·사적인·방해하지 않는
- NN/g 4축: 진지 82 / 격식 62 / 공손 78 / 객관 88
- 금지 인상: 마법 같은·말 많은·귀여운·미래적·과잉 확신 (AI 그라디언트·글로우 오브·의인화 카피 금지)

## 5. 형태 토큰

- radius 7~10px(톤 재설계 이후), spacing은 강제 그리드 없는 자유 연속형(캡션류 폰트만 정수 11/12px), motion 180ms
